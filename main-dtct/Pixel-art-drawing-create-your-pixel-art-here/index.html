Cara Menggunakan:

  • Pilih warna dari palet (16 warna) atau gunakan color picker untuk warna custom
  • Gunakan eraser () untuk menghapus pixel tertentu
  • Atur ukuran grid dengan slider untuk mengubah jumlah pixel (5x5 hingga 50x50)
  • Atur ukuran pixel dengan slider untuk mengubah ukuran setiap pixel
  • Klik atau drag pada kanvas untuk menggambar pixel
  • Gunakan tombol Bersihkan Kanvas untuk memulai dari awal
  • Gunakan tombol Export ke PNG untuk menyimpan gambar Anda
